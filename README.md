# Curated Rich OpenCores/FreeCores Outputs

This folder contains the cleaned outputs for projects that are useful for report writing and further dataset curation.

## Folder Contents

- `strongest_hdl_pdf_image_projects/`: full copied output folders for the strongest projects that contain HDL + PDF + image content.
- `all_rich_projects_index/rich_project_descriptions.json`: description metadata for all rich projects with HDL plus PDF or image content.
- `PROJECT_FIRST_RICH_PROJECTS.md`: project-first readable catalog with contents and direct source links.
- `REPORT_READY_ANALYSIS.md` and `.txt`: report-ready analysis summary.
- `rich_project_file_manifest.csv`: machine-readable file manifest for rich projects.
