1 Introduction
This IP core provides resource e’¼ācient ternary adders, i. e., adders with three inputs
performing s = x+y+z, for the Altera and Xilinx platforms. Resource e’¼ācient means
that they need exactly the same resources on modern FPGAs as two-input adders,

primitives (CARRY4, LUT6_2 and FDCE). It is suitable for all FPGAs providing 6-input

3 Implementation
Both implementations uses a carry save adder (CSA) tree with three inputs and a ’¼ünal
ripple carry adder as vector merging adder (VMA). One stage of full adders (FAs) is
used to realize a 3:2 compressor, i. e., the three input bit vectors are compressed to
two bit vectors which are obtained by the sum and carry outputs. A second stage of
FAs merges these two bit vectors to a single result.
