3 INTRODUCTION
3.1 BRIEF DESCRIPTION
PKT_CODEC_MK2 if a network interface component placed between the IP and the network.
It handles the clock domain crossing and the generation of the network depended packet headers.
Supported addressing modes include memory mapped, target id and raw where ip is responsible for
using correct network address.
